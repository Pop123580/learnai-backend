\# LearnAI Backend



AI-powered learning platform backend with OpenRouter integration.



\## Features



\-  AI Chatbot - Multi-language doubt solving

\-  Text Summarizer - AI-powered summarization

\-  PDF Summarizer - Extract and summarize PDF content

\-  Study Planner - AI-generated study schedules

\-  Exam Prep - Personalized exam preparation plans



\## Tech Stack



\- Node.js + Express

\- OpenRouter API (GPT-3.5)

\- Multer (File uploads)

\- PDF-Parse (PDF processing)
